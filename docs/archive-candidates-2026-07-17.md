# 미사용 기능 보관 후보 점검

요청서 원칙에 따라 아래 기능은 삭제하거나 이동하지 않았다. 대표 확인 후에만 `app/_archived/`로 이동하고 메뉴·라우팅에서 제외한다.

| 후보 | 현재 상태 | 현재 연결 | 제안 |
|---|---|---|---|
| `image-generator` | 사용 가능 | 개별 기능 목록과 고객 상세 홍보 앱에서 노출 | 퀄리티 보류가 확정되면 보관 |
| `variation` | 사용 가능 | 직접 라우트는 존재하나 주요 관리자 메뉴 노출은 낮음 | 우선 보관 후보 |
| `ai-trust-gap` | 사용 가능 | 개별 기능 목록과 고객 상세 홍보 앱에서 노출 | 실제 사용 빈도 확인 후 결정 |
| `video-convert` | 사용 가능 | 직접 라우트와 사진·영상 작업 영역에서 접근 가능 | 영상 운영 계획 확인 후 결정 |

## 이동 전 확인 사항

1. 기능별 최근 사용 여부와 저장 데이터 유무를 확인한다.
2. `lib/toolNav.ts`, 고객 상세의 홍보 앱 목록, 관리자 사이드바에서 링크를 함께 제거한다.
3. App Router는 `app/_archived/` 아래의 `page.tsx`도 라우트로 인식할 수 있으므로, 단순 폴더 이동 대신 프로젝트 루트의 `_archived/app/`처럼 빌드 대상 밖에 보관하거나 파일 확장자를 변경하는 방식을 사용한다.
4. 이동 후 `npm run build`로 기존 링크와 동적 import가 남지 않았는지 확인한다.

현재 추천 순서: `variation` → `video-convert` → `image-generator` → `ai-trust-gap`.
