# 개별 기능 홈 사이드바 진입점 보완

## 문제

개별 기능 카드 그리드는 `/admin/tools`에 구현되어 있지만, 관리자 사이드바의 `개별 기능` 섹션에는 개별 도구 링크만 노출되어 그리드 홈으로 직접 이동할 수 없다.

## 승인된 변경

- `개별 기능` 섹션의 첫 번째 항목으로 `개별 기능 홈`을 추가한다.
- 링크는 `/admin/tools`를 사용한다.
- 기존 사이드바의 아이콘, 활성 상태, 모바일 닫기 동작을 그대로 사용한다.
- 기존 개별 도구 목록과 CRM 컨텍스트 전달 방식은 변경하지 않는다.

## 완료 기준

1. 사이드바에서 `개별 기능 홈`이 개별 도구보다 먼저 보인다.
2. 항목을 선택하면 `/admin/tools`의 카드 그리드로 이동한다.
3. `/admin/tools`에서 해당 항목이 활성 상태로 표시된다.
4. 기존 개별 기능 링크는 그대로 동작한다.

## 검증

- TypeScript 검사
- 프로덕션 빌드
- 데스크톱 및 모바일 사이드바에서 링크와 활성 상태 확인
